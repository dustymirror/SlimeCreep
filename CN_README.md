# Slime Creep v1.13

![版本](https://img.shields.io/badge/version-1.13-blue)
![EuroPi](https://img.shields.io/badge/EuroPi-兼容-green)

**Slime Creep** 是一个为 EuroPi 开发的三通道混沌电压发生器，灵感来自蜗牛、黏液轨迹和缓慢的有机运动。

---

## 特性

- 三个独立的混沌发生器：**Creep**（粘行者）、**Ooze**（渗行者）、**Smear**（抹行者）
- 6 个 CV 输出（每通道 X/Y），0–10V 范围
- 每通道独立速度控制（0.3–10 倍率）
- 每通道独立深度控制（0.05–2.0）
- 外部 CV 耦合输入（Ain）
- DIN 触发输入（扰动混沌系统）
- OLED 屏幕实时显示轨迹
- 5 秒开机动画（蜗牛爬行：`;@-` / `;@ ~`）

---

## 安装

1. 通过 USB 将 EuroPi 连接到电脑
2. 将 `slime_creep.py` 复制到 EuroPi 驱动器
3. 重命名为 `main.py`（或通过菜单系统选择）

---

## 硬件接口

| 接口 | 功能 |
|------|------|
| **CV1 / CV4** | 通道 1（Creep）X / Y 输出 |
| **CV2 / CV5** | 通道 2（Ooze）X / Y 输出 |
| **CV3 / CV6** | 通道 3（Smear）X / Y 输出 |
| **K1** | 调节当前选中通道的**速度**，范围 0.3～10.0 |
| **K2** | 调节当前选中通道的**深度**，范围 0.05～2.0 |
| **Ain** | 外部 CV 耦合输入（0～5V），影响所有通道 |
| **B1** | 短按：切换到上一个通道 |
| **B2** | 短按：切换到下一个通道 |
| **B1+B2 同时按** | 软复位（重启程序） |
| **DIN** | 外部触发输入：扰动所有通道 |

---

## 三通道性格

| 通道 | 显示名称 | 基频 | 性格描述 |
|------|----------|------|----------|
| Creep | `Creep` | 0.0040 | 最慢，像蜗牛蠕动，适合超长周期调制 |
| Ooze | `Ooze` | 0.00065 | 中等速度，像黏液渗出，适合包络起伏 |
| Smear | `Smear` | 0.000025 | 最快，像涂抹痕迹，适合活跃调制 |

> 三个通道的基础速度不同，但都可以通过 **K1** 独立调节倍率。

## OLED 屏幕界面
┌─────────────────────────────┐
│Creep │ ← 当前选中通道
│ │
│ [X轨迹线] │ ← X 轨迹（连线）
│ [Y轨迹点] │ ← Y 轨迹（散点）
│ │
│X2.5 D0.78 D. A= │ ← 底部状态栏
└─────────────────────────────┘


### 底部状态栏说明

| 字段 | 含义 | 示例 |
|------|------|------|
| `X2.5` | 当前通道的速度值（0.3～10.0） | `X0.3`（最慢）～`X10.0`（最快） |
| `D0.78` | 当前通道的深度值（0.05～2.0） | `D0.05`（浅）～`D2.00`（深） |
| `D.` / `D*` | 扰动指示器：`D*` 表示收到 DIN 触发 | 触发后闪烁约 0.5 秒 |
| `A_` `A-` `A=` `A#` `A@` | Ain 耦合强度（_ 最小，@ 最大） | 外部 CV 越大，符号越密 |

## 旋钮控制

### Speed（K1）
- **范围**：0.3～10.0
- **手感**：线性响应，转动直接改变当前通道的速度
- **作用**：乘以该通道的基础速度，数值越大电压变化越快

### Depth（K2）
- **范围**：0.05～2.0
- **手感**：线性响应
- **作用**：控制混沌振子的“力度”，数值越大摆动幅度越强

> 💡 采用**增量控制**：只有转动旋钮时才会改变当前通道的值。切换通道后，新通道会保持之前的参数，不会被旋钮位置覆盖。三个通道可以独立调节，互不干扰。

---

## 开机动画

程序启动时播放约 **5 秒**的蜗牛动画：
- 蜗牛字符：`;@-` 和 `;@ ~` 交替（每 400ms 切换一帧）
- 蜗牛水平位置跟随当前选中通道的 **X 值** 变化
- 蜗牛垂直位置随机浮动（模拟爬行）
- 屏幕显示 `Slime Creep` 和版本号

动画结束后进入主界面。

---

## CV 输出范围

- 每个 CV 输出口输出 **0～10V**
- 每通道的 X 和 Y 是独立但相互耦合的混沌信号
- 三通道之间通过 **Ain** 输入耦合

---

## Ain 耦合说明

`Ain` 输入 0～5V 外部 CV 时：
- 该 CV 会影响**所有三个通道**的混沌行为
- 耦合强度在屏幕上用 `A_` → `A-` → `A=` → `A#` → `A@` 表示
- 耦合作用：使三个通道的运动相互拉扯/同步，产生更复杂的集体行为

---

## 扰动说明

当 **DIN** 接收到触发（上升沿）时：
- 所有三个通道同时受到一次随机扰动
- 屏幕上的 `D.` 临时变为 `D*`（持续约 5 帧）
- 扰动幅度与当前选中通道的 `depth` 成正比

用途：为混沌系统注入“意外”，产生不规则的突变。

---

## 按钮操作总结

| 操作 | 结果 |
|------|------|
| 短按 B1 | 切换到上一个通道（Creep ← Ooze ← Smear 循环） |
| 短按 B2 | 切换到下一个通道（Creep → Ooze → Smear 循环） |
| 同时按 B1+B2 | 软复位（重启程序，重新播放开机动画） |

> 注：长按不放不会触发切换，避免误操作。

## 典型使用场景

### 场景 1：慢速调制
- 选择 **Creep** 通道
- Speed 设为 0.5～1.5
- Depth 设为 0.3～0.8
- 将 CV1 或 CV4 接入 VCO 的 FM 输入，获得缓慢有机的漂移

### 场景 2：两个参数联动调制
- 将 CV1（X）和 CV4（Y）分别接入滤波器的 cutoff 和 resonance
- 两路信号相关但不相同，产生丰富的动态音色

### 场景 3：多通道随机 CV
- 三个通道的 CV 输出可以同时控制不同的模块参数
- 通过 Ain 输入 LFO 或序列器，让三个通道产生同步/异步的互动

### 场景 4：突发扰动
- 将随机触发信号（如时钟分频器输出）接入 DIN
- 混沌系统会不定期产生“抽搐”，打破原有轨迹

## 故障排除

| 现象 | 可能原因 | 解决方法 |
|------|----------|----------|
| 开机无画面 | 程序未上传成功 | 重新上传 EuroPi 固件和程序 |
| 旋钮无反应 | 转动太慢或阈值问题 | 检查 `THRESH` 常量（当前 0.008） |
| CV 输出无变化 | 通道未选中 | 按 B1/B2 确认选中通道 |
| 动画字符乱码 | OLED 驱动问题 | 确认使用 `;@-` 等 ASCII 字符 |
| DIN 触发无反应 | 信号电平不足 | 需要 0～5V 或更高触发信号 |
| 切换通道后参数被覆盖 | 旧版本 bug | 请升级到 v1.13 或更高版本 |
